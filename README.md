# Szentpéteri Méz

A website for Szentpéteri Méz, offering producer honey for sale directly from verified producer András Szentpéteri. We sell acacia honey, rapeseed honey, sunflower honey, and pollen in various packaging options. Special gift packaging is also available.

**Live Site:** [https://szentpeterimez.hu/](https://szentpeterimez.hu/)

## Tech Stack

- **Framework:** [Astro](https://astro.build/) - Static site generator for fast websites
- **UI Library:** [Preact](https://preactjs.com/) - Lightweight React alternative
- **Language:** JavaScript/TypeScript
- **Deployment:** Firebase Hosting
